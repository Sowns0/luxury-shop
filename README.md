 VIP Luxury Shop
Website bán hàng cao cấp được xây dựng bằng HTML, CSS, JavaScript thuần (Vanilla JS).
Project mô phỏng hệ thống thương mại điện tử với giỏ hàng, tìm kiếm sản phẩm, thanh toán và lưu trữ dữ liệu bằng LocalStorage.
Giới thiệu
VIP Luxury Store là một website bán sản phẩm cao cấp với giao diện sang trọng (tone tối – phong cách luxury).
Hệ thống hoạt động hoàn toàn phía client (Frontend Only).
